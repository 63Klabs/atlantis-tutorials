# Atlantis Platform In-Depth

The Alantis Platform is built and deployed using the very templates and scripts it provides to developers.

After completing the previous tutorials, you should have a better understanding of the templates, scripts, and configuration options Atlantis provides. By piecing these concepts and resources together, you have all you need to deploy your own internal Atlantis platform for use within your organization.

Even if you choose to continue using the publicly provided resources from 63Klabs, or even partitially adopt deploying your own custom templates, scripts, and starters alongside those maintained by 63Klabs, understanding how Atlantis deployments are orchestrated will allow you to adapt the workflows for other projects.
